🔌 Financial‑Data Sourcing (no‑code)

  1. Install PostgreSQL 16 + pgAdmin 4.
  2. Download CSVs: spy_daily.csv, eurusd_daily.csv, optional crypto CSVs.
  3. Create tables & import via pgAdmin Import Wizard.
  4. Peek with SQL (Query Tool):
-- recent SPY closes
SELECT * FROM spy_daily ORDER BY date DESC LIMIT 5;

-- join SPY ↔ EURUSD
SELECT s.date, s.close AS spy, e.close AS fx
FROM spy_daily s
JOIN eurusd_daily e USING(date)
ORDER BY s.date DESC LIMIT 5;

Week 2 · Data Wrangling & Hygiene

🧹 Tidy‑Data Checklist

  1. Drop duplicates & impossible prices (close <= 0).
  2. Timezone‑aware dates: pd.to_datetime(..., utc=True).
  3. Fill missing days: .asfreq('D').ffill().
  4. Merge on date; resample to weekly (Friday).
  5. Quick visual QA: line plot of closes, scatter of % returns.
  6. Data‑quality report: rows dropped, dupes, % missing.

Week 3 · Classical Econometrics (OLS)

⚙️ OLS Workflow Checklist

  1. Visualise & check stationarity (ADF test).
  2. Define regressand & regressors, add constant.
  3. Estimate with sm.OLS().
  4. Inspect R², p‑values, CI.
  5. Residual diagnostics: Jarque–Bera, Durbin–Watson, Breusch–Pagan.
  6. Iterate & document insights.
import statsmodels.api as sm

y = df['btc_excess']
X = sm.add_constant(df[['spy_excess', 'eth_excess']])
model = sm.OLS(y, X).fit(cov_type='HAC', cov_kwds={'maxlags':5})
print(model.summary())

# Function
def sharpe(r, rf=0.0):
    excess = r - rf
    return excess.mean() / excess.std()

# Class
class Strategy:
    def __init__(self, weights):
        self.w = weights
    def pnl(self, rets):
        return (rets * self.w).sum(axis=1)
